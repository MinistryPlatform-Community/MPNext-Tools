---
title: Subfolder README Template
type: meta
---

Copy this schema when writing a subfolder `README.md` (entry point for a domain).

```markdown
---
title: <domain>
type: index
domain: <slug>
---

## What's in this domain
One or two sentences.

## File map
| File | Purpose | When to read |
|------|---------|--------------|
| `file.md` | ... | ... |

## Code surfaces
| Path | Role |
|------|------|
| `src/...` | ... |

## Related domains
- `../other-domain/README.md` — why linked
```
